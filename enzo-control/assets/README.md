# Assets — ENZO CONTROL

Coloque aqui os arquivos visuais do projeto.

## Logo

O logo oficial da **Enzo Auto Service** deve ser salvo neste diretório com
o nome:

```
logo-enzo-auto-service.png
```

(Aceita também `.svg` se estiver disponível — nesse caso salvar como
`logo-enzo-auto-service.svg`.)

### Descrição do logo

Placa estilo retangular com cantos arredondados.
Fundo amarelo. Topo com listra verde/branca/vermelha (bandeira italiana).
Texto "ENZO" em preto grande e "AUTO SERVICE" em preto dentro de faixa preta
na parte inferior.

Cores oficiais derivadas do logo (usadas no tema dark do app):

| Nome       | Hex       | Uso                              |
| ---------- | --------- | -------------------------------- |
| Enzo Gold  | `#FDE015` | destaques, marca, CTAs primários |
| Enzo Red   | `#D31224` | alertas, urgências, status crítico |
| Enzo Green | `#009246` | sucesso, confirmações            |
| BG Dark    | `#0a0a0d` | fundo principal                  |
| BG Panel   | `#16161d` | cards e painéis                  |
| Text       | `#ececf0` | texto primário                   |
| Muted      | `#8a8a96` | texto secundário                 |

Esse esquema já está em uso no `app/index.html` do Track Day e será reaproveitado
para manter consistência visual entre os produtos da Enzo.
